# E-Commerce Sales Analysis

## Project Overview

Analyzed the Brazilian E-Commerce Public Dataset by Olist to uncover sales trends, customer behavior, delivery performance, and revenue insights using Python, SQL, Excel, and Power BI.

## Tools Used

- Python
- Pandas
- NumPy
- MySQL
- Excel
- Power BI

## Dataset

Brazilian E-Commerce Public Dataset by Olist (Kaggle)

## Project Workflow

### Data Cleaning
- Merged orders, customers, products, and order items datasets.
- Handled missing values and duplicates.
- Created Revenue, Delivery Days, Late Delivery, and Order Month features.

### SQL Analysis
- Total Revenue
- Monthly Revenue
- Revenue by Category
- Revenue by State
- Top 10 Customers
- Average Delivery Days
- Late Delivery Percentage

### Excel Analysis
- Pivot Tables
- Monthly Revenue Trends
- Top Customers Analysis

### Power BI Dashboard
- Executive Overview
- Sales Insights
- KPI Cards
- Revenue Analysis
- Customer Analysis

## Key Insights

- Identified top-performing product categories.
- Analyzed monthly sales trends.
- Measured delivery performance.
- Found top revenue-generating customers.
- Compared revenue across states.
